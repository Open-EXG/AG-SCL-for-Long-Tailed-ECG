# README figure sources

The figures in this directory are author-provided derivatives of figures from
the final AG-SCL manuscript. Scientific content, labels, plotted values, error
bars, and legends were not altered.

| Public asset | Manuscript source | README use | Deterministic processing |
| --- | --- | --- | --- |
| `ag-scl-overview.png` | `Fig/frame_figure.png` | Method overview | Composited on white, cropped to non-white content with padding, resized to 2000 px width, metadata removed |
| `angular-gaussian-geometry.png` | `Fig/Distribution.png` | Covariance concept | Composited on white, cropped to non-white content with padding, original scale retained, metadata removed |
| `per-class-results.png` | `Fig/fig_per_class_bar.png` | Paper results | Composited on white, cropped to non-white content with padding, resized to 1600 px width, metadata removed |
